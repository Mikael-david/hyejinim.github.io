Samsung Smart TV

Improvement of Smart TV'14 UI and developed an UI general guidelines



CLIENT Samsung Electronics
ROLE Guideline, UI Design
DURATION Jun 2014 - Dec 2014


We improved the user interface of the Samsung Smart TV’14 and developed an integrated UI design guidelines for future releases. At that time, multiple teams at Samsung had used different design principles for different components of the interface. We grouped UI elements based user scenarios and derived common design principles focusing on providing improved user experience.
